## 🔧 Customization

### Colors
Modify CSS custom properties in `:root`:
```css
:root {
  --primary-500: #3b82f6;
  --neutral-800: #1e293b;
  /* ... other colors */
}
```

### Typography
Update font variables:
```css
:root {
  --font-family: 'Your Font', sans-serif;
  --font-size-base: 1rem;
  /* ... other typography */
}
```

### Animations
Adjust timing functions:
```css
:root {
  --transition-normal: 300ms ease-in-out;
  --transition-slow: 500ms ease-in-out;
}
```
